１　BSEコンバーターの概要
BSEコンバーターは、音声点字携帯情報端末「ブレイルセンスポラリス」、「ブレイルセンスポラリスミニ」、「ブレイルセンスシックス」、「ブレイルセンスシックスミニ」専用のアプリケーションです。
BASE形式の点字データ（.bse）を、BRL/BRF形式に変換できます。
これにより、センスシリーズのノートパッドを使って、BASEファイルの内容を確認できるようになります。
なお、本アプリを使用するには、センスシリーズのファームウェアを最新に更新する必要があります。
本アプリを使用できる最初のファームウェアは、以下の通りです。
・ポラリス/ポラリスミニ：57921(20231214版)
・シックス/シックスミニ：57043(20231019版)
なお、本マニュアルは、センスシリーズの基本的な操作方法を習得済みの読者を想定して記述しています。

２　インストールとアンインストール
本アプリのインストールおよびアンインストール方法は、有限会社エクストラが公開している追加アプリケーションとまったく同じです。
具体的な手順につきましては、有限会社エクストラが公開している情報をご参照ください。
【参考】有限会社エクストラWebサイト内　ブレイルセンスシリーズのダウンロードページ
https://www.extra.co.jp/download_w/sense_dl.html
本アプリをインストールすると、センスシリーズのメインメニューに[サードパーティ]が追加され、その中に[BSEコンバーター]が表示されます。

３　使用方法
アプリを起動すると、「BSEファイルを選択」と表示され、ファイル選択のダイアログボックスが開きます。
センスシリーズのファイル管理と同様の操作で、変換したいファイルを選択してください。
次に、「変換形式 brl」と表示されます。
ここでSpaceキーを押すと、[brl]と[brf]の選択を切り替えることができます。
[brl]を選択すると、ページ行の内容が失われ、各段落の末尾にのみ改行コードが入ります。
ただし、BASE形式の仕様上、意図した箇所に改行が挿入されない可能性があります。
[brf]を選択すると、ページ行の内容が維持され、各行の末尾に改行コードが入ります。
選択したらEnterキーを押してください。
次に、ファイルの保存先を既定値から変更するかどうかを尋ねるメッセージが表示されます。
既定の保存先は、変換元ファイルの拡張子を「brl」や「brf」に変更したものです。
変更する必要がなければ、そのままEnterキーを押してください。
保存する場所やファイル名を変更したい場合には、Spaceキーで[はい]に切り替えてからEnterキーを押してください。
[はい]を選択すると、ノートパッドなどと同様の「名前を付けて保存」ダイアログが表示されます。
以上の操作で、ファイルの変換が完了します。
最後に、「続けて別のファイルを変換しますか？ はい」と表示されます。
このままEnterキーを押すと、アプリ起動直後の状態に戻り、別のファイルを変換することができます。
Spaceキーで[いいえ]に切り替えてEnterキーを押すと、アプリを終了してメインメニューに戻ります。
